1. List:
   Set is a collection of unique values
   It is Mutable, Ordered
   It is enclosed with []

2. Tuple:
  Tuple is similar as List but this is Immutable
  It is enclosed with ()

3. Set:
   set is a collection datatype used to stored multiple unique values
   set is unordered
   It is enclosed with {}

4. Dictionaries:
   Dictionaries are key-value pair type

5.comprehensions:
  comprehensions make the code in one line for the list, set, dictionaries
  It is used to reduce code length, improve readability

6. json(javascript object notation)
   It is a lightweight format used to store and exchange data.
